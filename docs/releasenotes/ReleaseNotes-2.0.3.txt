Release Notes
xTuple Time & Expense (xtte)
Version 2.0.3
April 13, 2011
==================================

This is the 2.0.3 patch release of the xTuple Time & Expense
package, a free and open source extension package for xTuple ERP
(PostBooks, Standard, and Manufacturing Editions).

The Time & Expense package (xtte) is available under the same
Common Public Attribution License (CPAL) as the PostBooks Edition
of xTuple ERP (www.xtuple.com/CPAL). For more information on
xTuple's extension package architecture, please see
www.xtuple.org/developer-zone.

==================================

The following bug fixes have been added to the
package since 2.0.2. Additional detail for each item listed below
may be found on our community website (www.xtuple.org). 

Bug Fixes:

* Time sheet switched from expense to time can be marked as "prepaid." [#14360]
* TE prevents saving new item [#13791]
* Print report of Expense Sheet doesn't display the text clearly [#12570]
* Project labels not disabled on configureCRM [#13653]
* Item Number field doesn't refresh on shuffling between tasks in sheet screen [#13723]
